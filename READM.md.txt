Projet personnel de création d'article sécurisé en PHP.

Prérequis

PHP
Composer
Symfony CLI

Installation

- Clonez le dépôt : git clone url_du_dépôt
- Installez les dépendances : composer install
- Configurez les variables d'environnement dans le fichier .env selon le besoin
- Lancer le serveur : symfony server:start

Ouvrez votre navigateur et allez à http://localhost:8000. Un route admin est également disponible.
